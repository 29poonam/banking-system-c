# banking-system-c
Beginer friendly C programming mini project
This project is a simple Banking system developed using c programming language. It is menu driven console application that allows user to perform simple banking operations.
##features
-Create new account
-Deposit money
-Withdraw money
-Check account balance
-Exit the system
## Concepts used
-Functions
-Loops
-Switch case
-Conditional statements
## Purpose
-This project was created to practice core c programming concepts and improve problem solving skill.
